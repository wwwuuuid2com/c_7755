<h3>源码简介：</h3>
<p>这是一款主打字体设计,符号组合等多模板功能的一款微信小程序源码<p>
<p>内含多种功能,比如:<p>
<p>花式字体设计<p>
<p>表情字体组合设计<p>
<p>翅膀字体组合<p>
<p>火星文一键生成<p>
<p>符号库<p>
<p>空白名称<p>
<p>彩色网名等等<p>
<p>还有很多,所以小编就不一一介绍了<p>
<p>通过用户的自定义输入和选择,可以让用户生成各种字体,或者组合各种名字文案等等<p>
<p>另外小编还给添加了多样化的流量主模式<p>
<p>大家只需要替换对应的流量主ID即可<p>
